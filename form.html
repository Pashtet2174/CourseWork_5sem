Руководство пользователя

1. Начало работы и авторизация

При запуске программы откроется окно авторизации.

1
Введите ваш логин и пароль в соответствующие поля.
2
Нажмите кнопку входа для доступа к главной панели.

2. Работа с таблицами

В системе доступны два основных типа таблиц:

  • Справочники: общие таблицы базы данных, доступные всем пользователям.
  • Мои таблицы: персональный набор таблиц, сформированный согласно вашим правам доступа.

Операции с данными (CRUD)

Нажмите на название таблицы, чтобы открыть вкладку управления.

+
Добавление: Нажмите Добавить, введите данные в диалоговом окне и нажмите Сохранить.
×
Удаление: Выделите нужную запись кликом мыши, затем нажмите Удалить.
Примечание: Запись не может быть удалена, если она связана с данными в других таблицах.
Q
Поиск и сортировка: Используйте поисковую строку и кнопку Найти либо нажмите на заголовок столбца для сортировки.

3. Вкладка «Документы»

Раздел предназначен для работы с SQL-запросами и формирования отчетов.

  • Для стандартных отчетов используйте готовые кнопки вызова SQL-запросов.
  • Для произвольной выборки введите свой запрос в текстовое поле и нажмите Запустить.
  • Результат можно выгрузить, нажав Сохранить в файл (потребуется указать путь сохранения).

4. Аналитика

В этом пункте визуализируются данные в виде графиков.

↻
Для актуализации данных в реальном времени включите автоматическое обновление кнопкой в левой части экрана.

5. Настройки и безопасность

Раздел «Разное»

  • Смена пароля: Введите старый пароль, затем новый и его подтверждение. Нажмите Сменить пароль.
  • Настройки приложения: Выберите подпункт «Настройки» для изменения интерфейса или параметров под свои предпочтения.
Для завершения сеанса работы используйте кнопку «Выход» в правом верхнем углу экрана.

6. Помощь

Информацию о версии программного обеспечения и разработчиках можно найти в меню Справка → О программе.
